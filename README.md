# Toward a Shared, Decoupled Evidence Layer for AI-Era Science

**[Read the published paper](https://allenneuraldynamics.github.io/decoupled-evidence-layer/)**

This repository contains the MyST source, figures, structured figure data, and bibliography for a consensus perspective on interoperable, independently citable evidence records. Discussion and contributions are welcome through GitHub issues and pull requests.

## Local development

```bash
npm install
npm install -g mystmd
myst start
```

Build the static site with `myst build --html`.
